Creating a simple array with elements

How we create a new array with the assigned values? see the example below:

int[] javaArray = {1, 2, 3, 4, 5}

Is to easy, only following this steps:

  • Step 1 - Declare the type of the array
  • Step 2 - Putting a name of the array
  • Step 3 - Insert your values

Declare the type of the array

To declare the type of your array, you need to known some types that java accept as:

  • int: for the integers numbers
  • double/float: for the numbers with floating point
  • bool: for true or false
